## 基本功能
支援場景、人物、物品、人物頭像、人物立繪、物品圖標、對話選項、支線劇情
## 額外功能
場景動畫、心情變換、音效播放、遊戲存檔
## 遊玩方式
1. 在終端機執行`make`
2. 在`final_project`目錄下執行 `./bin/interactive_novel_engine`
